سياسة الخصوصية

هذه الصفحة تشرح كيفية استخدام بياناتك عند إرسال طلب حجز عبر موقع Pearly Smile Dental Center.

ما الذي نجمعه

عند إرسال نموذج الحجز، قد نجمع: الاسم، رقم الهاتف، الطبيب/الخدمة المختارة، التاريخ/الوقت، وملاحظات إضافية.

كيف نستخدم البيانات

  • للتواصل معك لتأكيد الموعد أو تعديل الوقت.
  • لتنسيق الجدول مع الطبيب والفرع.
  • لتحسين تجربة الحجز وجودة الخدمة (بشكل عام وبدون نشر بياناتك).

مشاركة البيانات

لا نبيع بياناتك. قد يتم الوصول إليها من قبل موظفي العيادة/الإدارة فقط لتنفيذ طلب الحجز.

الاحتفاظ بالبيانات

نحتفظ بالبيانات للفترة اللازمة لإدارة المواعيد والالتزامات التنظيمية. يمكنك طلب حذف بياناتك عبر التواصل مع العيادة.

حقوقك

  • طلب معرفة البيانات التي نحتفظ بها عنك.
  • طلب تصحيح بيانات غير دقيقة.
  • طلب حذف البيانات (عند الإمكان).
